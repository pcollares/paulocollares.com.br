Um script simples para testar endpoints de uma API.

O Código pode ser encontrado no meu Github:

https://github.com/pcollares/api-test

Este script basicamente acessa vários endpoints pelo curl e testa a sua disponibilidade, gerando um relatório ao final de quantos foram acessados com sucesso.

O script testa apenas requisições GET.

Altere a variável 'URL_BASE' para a URL base da sua API;

O arquivo 'sample' deve ser editado com os endpoints que deseja testar. Ou criado outro arquivo com a mesma estrutura do sample.

Exemplo de uso:

$ api-test.sh sample
