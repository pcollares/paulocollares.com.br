Listo aqui algumas dicas úteis para se configurar o Glassfish:

Alterar da porta 8080 para 80

Navegar até 'Configurations > server-config > Network Config > Network listeners > http-listner-1' alterar a porta de 8080 para 80

Alterar da porta 8181 para 443 (https)

Navegar até 'Configurations > server-config > Network Config > Network listeners > http-listner-2' alterar a porta de 8181 para 443

Habilitar a compressão

Navegar até 'Configurations > server-config > Network Config > Network listeners > http-listner-x > HTTP' colocar 'Compression' em 'force'

Habilitando a criação de links simbólicos (Linux)

Esta dica é útil para, por exemplo, disponibilizar um diretório de imagens muito grande dentro do servidor de aplicação. Para isso é necessário criar um link simbólico dentro do diretório 'docroot' apontando para o diretório real, basta habilitar essa opção no Glassfish:

Navegar até 'Configurations > server-config > Virtual Servers > server' adicionar em 'additional properties' a chave: allowLinking / valor: true

E depois criar um link no diretório docroot apontando para a pasta de imagens:
ln -s /home/usuario/imagens /opt/glassfish-4.1/glassfish/domains/domain1/docroot/imagens
