Alguns anos atrás comecei a desenvolver esse sistema para auxiliar nas projeções da minha igreja, não consegui finalizar e decidimos por uma solução comercial.

Decidi então, disponibilizar o código para quem quiser usar , está hospedado no Github, pode ser usado sem restrições e modificado.

Faça download ou clone no Github

Tecnologia utilizada

  • Java + JavaFX

Funcionalidades

  • Apresentar músicas
  • Exibir Bíblias completas
  • Exibir vídeos
  • Preparar o cronograma do culto
  • Exibir a logo da igreja, sobrepondo qualquer apresentação
  • Exibir um fundo preto, sobrepondo qualquer apresentação

Como usar

Basta executar o projeto, algumas configurações devem ser modificadas manualmente no arquivo config.properties.

O diretório recursos deve conter os videos, fundos, logos, músicas e bíblia. Detalhes de cada recurso são descritos abaixo:

Bíblias

No repositório contém a Almeida Atualizada como exemplo, deve ser descompactada no diretório de bíblias dentro de recursos.

Cada Bíblia deverá ter seu próprio diretório, e dentro dele, um diretório para cada livro, e dentro deste, arquivos de texto nomeados com os números dos capítulos. Cada arquivo de capítulo contém uma linha para cada versículo.

Fundos

O diretório de fundos contem os arquivos de vídeo que será apresentado como fundo das letras das músicas e da Bíblia. No arquivo config.properties é possível informar qual arquivo de fundo será usado para a Bíblia e para as músicas.

Imagens

Não implementado nessa versão.

Logos

O diretório de logos contem os arquivos de vídeo que será apresentado como logo quando esse botão for pressionado no sistema. No arquivo config.properties é possível informar qual arquivo de logo será usado.

Músicas

O diretório de músicas contem as músicas que serão apresentadas no sistema. Cada música deve estar em um arquivo de texto nomeado com o nome da mesma. Seu conteúdo deve seguir esse formado:

TITULO = [Nome da música]
ARTISTA = [Autor da música]
FUNDO = [Nome do arquivo de fundo que será usado para apresentar a mesma]

[Parágrafo 1]

[Parágrafo 2]

[Parágrafo n]

Vídeos

O diretório de vídeos contem os vídeos que podem ser apresentados no sistema.
