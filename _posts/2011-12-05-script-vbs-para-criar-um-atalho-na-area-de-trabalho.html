Esse Script é muito útil para se você quiser divulgar um link na área de trabalho das estações do seu domínio, ele cria um atalho com a possibilidade de um ícone personalizado, que abre uma página na Internet, ou na Intranet no navegador padrão da máquina.

Dependências:
Um script .bat configurado para chamar esse arquivo, sendo executado no logon do usuário.

Código

set WshShell = WScript.CreateObject("WScript.Shell")
strDesktop = WshShell.SpecialFolders("Desktop")

set oUrlLink = WshShell.CreateShortcut(strDesktop & "\NOME DO LINK.lnk")

oUrlLink.TargetPath = "http://www.sua_url.com"

oUrlLink.IconLocation = "CAMINHO PARA UM .ico"

oUrlLink.Save
