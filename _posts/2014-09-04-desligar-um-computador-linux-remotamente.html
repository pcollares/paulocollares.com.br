Este simples script desliga um computador linux remotamente.

Primeiramente instale o sshpass na maquina que executará o script:

#apt-get install sshpass

Também será necessário instalar o ssh na máquina remota:

#apt-get install ssh

Script:

#! /bin/bash

host="192.168.0.1"
senha="senha_de_root"

sshpass -p $senha ssh root@$host 'shutdown -h now; exit'

[]`s
