Estes são três passos simples para manter seu código java limpo, claro e mais fácil para os outros desenvolvedores entenderem no Netbeans, são algumas boas práticas de programação:

Javadoc
Esse não é exclusivo do netbeans, mas é válido mencionar :p
Antes de realizar o commit, escreva o javadoc das novas classes, assim como seus métodos. Isso ajudará aos outros desenvolvedores e a você mesmo.

Importações
Evite importações desnecessárias dentro das classes, corrija as importações com o atalho ctrl+shift+i

Identação
Para uma melhor visualização, idente e formate corretamente o código, use o atalho alt+shift+f
