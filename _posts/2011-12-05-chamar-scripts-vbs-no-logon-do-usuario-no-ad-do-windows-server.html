Scripts .vbs são bastante úteis para manipulação de algumas tarefas em uma rede com um AD.

Primeiramente crie um arquivo .bat em C:/Windows/SYSVOL/domain/scripts/ nomeie ele de logon.bat, por exemplo, dentro dele chame os seus scripts .vbs que deverão estar nesta mesma pasta, exemplo:

mensagem.vbs
mapear.vbs
links.vbs

Abra o gerenciador de "Usuários e Grupos do Active Directory" em Ferramentas Administrativas do AD, selecione todos os usuários e clique com o botão direito > Propriedades > Perfil. Em Scripts de Logon escreva "logon.bat", que é o nome do arquivo que criamos anteriormente.

Exemplos de Scripts .vbs
- Script .vbs para mapeamento automático de unidades de rede por grupo no AD
- Script .vbs para exibir uma caixa de alerta para todos os usuários do AD
