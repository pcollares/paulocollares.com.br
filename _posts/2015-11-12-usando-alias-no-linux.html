Um alias é um atalho para um comando que pode ser chamado em qualquer momento no terminal, útil para chamar rotinas que dependeriam d entrar em um diretório longo, por exemplo. Para usar o alias basta digitar o seguinte comando:

$ alias home='cd /home/pcollares'

Agora, basta digitar 'home' para que seja executado este comando. Para visualizar todos os alias criadas, digite alias, e para remover use o unalias.

Alias permanente

O comando alias é valido apenas durante a sessão atual, para isso devemos garantir que seja adicionado a cada nova sessão, para isso,  vamos adicionar nossos alias no arquivo ~/.bash_aliases (um arquivo oculto que se encontra em seu diretório pessoal, crie ele caso não exista).
