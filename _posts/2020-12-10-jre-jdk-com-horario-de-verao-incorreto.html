A JRE/JDK mantém uma lista com os fusos e horários de verão dos países. Se estiver usando uma JDK antiga, essa lista pode estar desatualizada de acordo com os últimos decretos sobre o fim do horário de verão, para corrigir esse problema, devemos atualizar esta lista conforme descreve os passos a seguir:

1. Efetue o download da última versão do banco de dados de TimeZone: https://www.iana.org/time-zones (Data Only Distribution)

2. Efetue o download da ferramenta de atualização dessas informações: https://www.oracle.com/technetwork/java/javase/downloads/tzupdater-download-513681.html (Java SE TZUpdater)

3. Extraia do arquivo compactado o arquivo tzupdater-2.3.2.zip

4. Copie os arquivos tzupdater.jar e o tzdata2020d.tar.gz para um diretório temporário qualquer

5. Execute: java -jar tzupdater.jar -v -f -l file:tzdata2020d.tar.gz
